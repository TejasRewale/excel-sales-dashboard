# Sales Performance Dashboard Using Excel	
	
# Project Objective	
The objective of this project is to analyze and visualize sales data using Microsoft Excel. This dashboard helps in tracking overall sales performance, identifying top-performing products, monitoring monthly trends, and making data-driven business decisions using interactive visuals.	
KPIs (Key Performance Indicators) Used	
# KPI	
  -Total Target	
  -Total Quantity	
  -Total Profit	
  -Total Sales	
  -Department Wise Sales	
  -Region Wise Sales	
  -Top 5 Product	
  -Monthly Sales Trend	
	
# Tools & Features Used in Excel	
  Tool / Feature	Purpose in Dashboard
  Pivot Tables-	To summarize large sales data quickly
  Pivot Charts-	To create dynamic charts from pivot data
  Slicers-	To filter data by Month / Product / Category
  Formulas-	SUM, AVERAGE, COUNT for KPI calculations
  Conditional Formatting-	To highlight important values
  Tables-	To manage and clean raw data
